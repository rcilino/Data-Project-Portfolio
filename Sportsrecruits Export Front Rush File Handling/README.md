# Sportsrecruits Export > Front Rush File Handling

Python script needs to run with two folders present:
- For Processing-Sportsrecruits Exports
- Output-FrontRush Import Files

Files need to be CSV, direct exports from Sportsrecruits should work properly without editing.

If the Gender column should be populated, include "Men" or "Women" in the Sportsrecruits export CSV file name.
